# 言语理解与表达

### 必考题型

1. 片段阅读，概括主旨 or 判断细节
2. 语句表达，选语句（排序，填空，接语）
3. 逻辑填空，选词（成语，实词，成语和实词）



(1):包括中心理解题和细节判断题。 

(2)语句表达:包括语句排序题、语句填空题和接语选择题(接语选择题如给你一段文字，问作者接下来最有可能讲的是......)。 

(3)逻辑填空:包括实词填空、成语填空和混搭填空。
