# 2.1 Getting a Git Repository

有两种方式创建Repo：

1. 将本地文件转化为Repo
2. 克隆远程Repo到本地

以macOS系统作为示范。

## 将本地文件夹转化为repo

打开Terminal，并切换路径到本地文件夹

```
$ cd /Users/user/my_project
```

初始化本地文件夹

```
$ git init
```



## 克隆远程repo到本地

在github上找到想要克隆到本地的Repo

输入命令，并拉到本地仓库

```
$ git clone https://github.com/libgit2/libgit2
```
